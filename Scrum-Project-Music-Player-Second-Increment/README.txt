This is the first increment of the Media Player project using Scrum Development. To run, please open the "main.py" file.

Please Make sure to download the following packages before running the application.
- Python 3
- PySide2

To install Pyside using cmd, run the following commands:
- pip install PySide2


Note: if the application does not play videos, please make sure you have the proper video codexes installed in your PC.
